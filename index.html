Simulador de Compiladores

Este proyecto busca exponer el funcionamiento de los procesos de analisis lexico y sintactico de los compiladores: desde el flujo de caracteres hasta la evaluación semántica del AST.

bash — python main.py
➜ python main.py "(12 + 8) * 2 ^ 3"
[...] Iniciando Pipeline de Compilación...
[ * ]
├── [ + ]
│   ├── [ 12 ]
│   └── [ 8 ]
└── [ ^ ]
    ├── [ 2 ]
    └── [ 3 ]
VALOR CALCULADO: 160.0

Arquitectura del Proyecto

Lexer

Convierte el código fuente en una secuencia de tokens con significado.

Parser

Valida la sintaxis mediante gramática de descenso recursivo.

AST Nodes

Estructura jerárquica que representa la semántica de la expresión.

Evaluación

Recorre el árbol recursivamente para obtener el resultado final.

Exploración profunda de clases y métodos del sistema.
